# Runbooks

Operational guides for repeatable project workflows.

New users should start with the Easy Mode CLI documented in the platform quick
starts:

```text
python clm.py init
python clm.py check
python clm.py crawl "collect product titles and prices" mock://catalog --output dev_logs/runtime/mock_result.json
```

- `QUICK_START_LINUX_MAC.md`

1. `QUICK_START_WINDOWS.md` or `QUICK_START_LINUX_MAC.md`
