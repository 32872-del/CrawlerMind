## Requirements

Implement a deterministic MVP that can:

1. Normalize a `BrowserContextConfig` into a fingerprint profile:
   - user agent
   - viewport
   - locale
   - timezone
   - color scheme
   - JavaScript enabled
   - proxy present or absent, redacted
   - storage state present or absent, redacted
2. Produce consistency findings:
   - mobile UA with desktop viewport
   - desktop UA with tiny/mobile viewport
   - locale/timezone mismatch heuristics
   - missing UA when a custom profile is expected
   - proxy configured but timezone/locale left default
3. Return a serializable report:
   - profile summary
   - findings
   - risk level: low/medium/high
   - recommendations
4. Add unit tests with no real browser launch.
