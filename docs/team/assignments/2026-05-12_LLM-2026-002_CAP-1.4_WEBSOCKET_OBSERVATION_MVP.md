## Requirements

Implement:

1. Data classes for WebSocket connection/frame summary:
   - URL
   - frame direction: sent/received
   - text/binary marker
   - bounded preview
   - byte length
2. A result object with connections, frames, errors, and status.
3. Helper functions to normalize frame payloads without leaking huge payloads.
4. A mocked Playwright event handler test path for `page.on("websocket", ...)`.
5. Redaction or bounding for frame previews.
