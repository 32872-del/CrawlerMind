## CI Limitations

1. **No browser smoke**: Playwright browsers are not installed in CI. Browser
   smoke tests are gated by `AUTONOMOUS_CRAWLER_RUN_BROWSER_SMOKE` env var
   and skip automatically when unset.
2. **No real network crawls**: CI runs the standard unit/integration suite only.
   Real-site training scripts (`run_training_round*.py`, `run_baidu_hot_test.py`)
   are not run in CI.
3. **No LLM tests**: No API keys in CI. LLM advisor tests use fake clients.
4. **Optional deps may fail**: `curl_cffi` and `botasaurus` have C extensions
   that may not build on all CI runners. If they fail, the test suite still
   runs for modules that don't depend on them.
